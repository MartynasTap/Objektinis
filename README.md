# Objektinis
V0.1 programos versija,kuri buvo pildoma is programos Pradine. Pakeitimai nuo Pradine versijos - vartotojas gali pasirinkti, ar nori, kad duomenys butu skaitomi is failo. Isvedami duomenys surusiuoti pagal studentu vardus ir sulygiuoti pagal pateikta pavyzdi reikalavimuose.
Papildyta isimciu valdymu tikrinti, ar atidaromas failas egzistuoja, ar eilute, kuria norime paimti, nera tuscia. Taip pat tikrinant ar ivestas pazymys yra tarp 1 ir 10.
